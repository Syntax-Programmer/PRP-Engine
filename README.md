# PRP Engine

An engine made for Pure Raw Performance:

PRP Engine is an experimental game engine written in C focused on performance, modular architecture, and low-level control.

The project is primarily a learning and research effort exploring game engine architecture, data-oriented design, and graphics programming.

---

## Overview

PRP Engine explores how modern engines can be built from scratch using simple and efficient systems.

Current development focuses on:

- Entity Component Systems (ECS)
- OpenGL/Vulkan rendering pipeline
- data-oriented architecture
- modular engine systems
- performance-focused design

---

## Features

Current and planned engine features include:

- Entity Component System (ECS)
- OpenGL rendering
- SDL window and input handling
- modular subsystem architecture
- custom math utilities
- resource management
